JavaScript 開發者年會

今年，JavaScript 開發者年會 (JavaScript Developers Conference, 簡稱 JSDC) 打破了壁壘，整合來自獨立開發者、企業、組織及機構的科技力量。 我們致力使知識全球化，並且讓參與者重新認識 JavaScript 。 我們的新思維將更進一步改善架構、強化團隊及最佳化開發流程。
